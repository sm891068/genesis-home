# 舊版內容盤點與淘汰規則

v0.1 重建不是把舊版全部刪掉，也不是全部照搬。舊版內容會被當作素材庫，經過盤點後再決定進入新版本的方式。

## 已盤點的舊版內容

### 角色與職業

舊版已有 26 位角色，職業包含：

- 戰士 (FIGHTER)
- 保鏢 (BODYGUARD)
- 刺客 (ASSASSIN)
- 槍手 (GUNNER)
- 醫生 (DOCTOR)
- 狙擊手 (SNIPER)
- 女王 (QUEEN)
- 秘書 (SECRETARY)

這些分類在 v0.1 中不直接作為正式分類，而是轉成「舊版原型分類」。

## 新版處理方式

### 保留

可以保留的內容：

- 角色稀有度概念。
- 角色能力值概念。
- 性格影響對話的概念。
- 父親遺產與阿福引導者定位。
- 女王型與秘書型的核心吸引力。

### 改寫

需要改寫的內容：

- 女王型：改為統御型角色，不只是後宮角色。
- 秘書型：改為管理型 / 顧問型角色，不只是輔助角色。
- 戰士、保鏢、槍手、狙擊手：整合為武鬥型，避免職業太碎。
- 醫生：整合為支援型，但保留醫療與降低損失能力。
- 後宮：先改成信任/關係系統，再逐步擴充親密劇情。
- 吃卡：暫不進 v0.1 正式規劃，避免角色工具化過重。

### 淘汰或暫停

暫停內容：

- 過早的大量抽卡。
- 完整吃卡升級。
- 過度分散的職業命名。
- 沒有世界觀支撐的角色。
- 只為稀有度而存在的角色。

## v0.1 新角色分類

新版角色分兩層：

### 組織階級

- 會長 / 組長
- 若頭
- 顧問
- 幹部
- 兄弟分
- 若眾
- 外部協力者

### 功能定位

- 統御型
- 管理型
- 武鬥型
- 情報型
- 支援型
- 社交型

## 日本黑幫參考方向

v0.1 以日本黑幫 / 極道風格作為起點，重點是：

- 家族式階級。
- 義理與人情。
- 組織面子。
- 地盤秩序。
- 舊派與新派衝突。
- 父親留下的規矩與底線。

遊戲會使用架空城市與架空組織，不直接複製現實團體，也不提供現實犯罪操作細節。
