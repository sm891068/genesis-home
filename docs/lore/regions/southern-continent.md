# 南大陸

## 大陸概述

南大陸是帕拉維亞人界的重要大陸之一，整體由四大境域構成：

- 東境
- 西境
- 北境
- 南境

四境並非單純行政分區，而是受到地形、氣候、族群活動與魔力流向共同塑造的地理區域。

## 四境分區

### 東境

東境以原始森林、河谷、海岸與古代遺跡為主要地貌。

格拉姆地區位於南大陸東境，是東境最具代表性的自然與星紋交會地帶之一，包含格拉姆森林、格拉姆遺跡、海岸與外海區域。

古望塔也位於東境格拉姆森林之巔，因此東境同時具備自然、遺跡、星紋觀測與邊境防務意義。

### 西境

西境尚待補齊。

可作為後續國家、荒原、商路、軍事邊境或魔族接觸區的設定空間。

### 北境

北境是南大陸與北方地理系統接壤的邊界地帶。

北境上方橫亙著紅土山脈，形成阻斷南北大陸交通的巨大天然屏障。

### 南境

南境以海岸、半島、港灣與南方海域為主要地理特徵。

歌碧半島預計位於南境，是南大陸向海洋延伸的重要半島地帶，可作為海洋國度、港口城市、封印神柱、南方航路或深淵海域威脅的設定核心。

### 歌碧半島

歌碧半島是南境的重要地理節點，位於南大陸向外海延伸的邊界區域。

其後續可承載以下設定：

- 南境主要港口與海上貿易路線。
- 連接外海、群島與蒼炎外島的航道節點。
- 與海洋循環、潮汐魔法或伊娃娜信仰相關的地區文化。
- 可設置封印神柱、海底遺跡或深淵海域裂縫。
- 作為南境冒險者、海軍、商會與異端勢力交會之地。

## 紅土山脈

紅土山脈位於南大陸北境上方，是阻斷南北大陸交通的關鍵地形。

其主峰高度約達 3000 公尺級，山體由紅土、裸岩、針葉森林與高山地形交錯構成。

### 地貌特徵

- 紅土帶：山腳與低海拔區域可見大面積紅土，土壤富含礦質與古老魔力沉積。
- 森林帶：中低海拔分布森林，形成天然遮蔽與魔獸棲息地。
- 山脈帶：高海拔地形陡峭，氣候嚴酷，構成天然阻隔。
- 高峰區：主峰約 3000 公尺級，常年雲霧與強風籠罩。

### 戰略意義

- 阻斷南北大陸交通，使陸路往來困難。
- 壓縮南北商路，使通行必須依賴特定山口、古道或海路。
- 成為軍事防線與勢力邊界的天然基礎。
- 可能蘊藏古代礦脈、遺跡或封印設施。

## 地理定位

南大陸的地理核心不是單一平原，而是由森林、山脈、海洋與遺跡共同構成的複雜地形。

其中紅土山脈是大陸北側的阻隔線，格拉姆地區則是東境的自然與星紋核心。
