# 光輝系統與符文之路

帕拉維亞的裝備進化與魔法導通，根植於被稱為「光輝符文之路」的古老技術。

此路象徵力量的延續與靈魂的共鳴，亦是鍛造師、冒險者與神祇意志交織的命運之途。

它不僅是強化之法，更是文明對星辰語言的理解與再現。

## 一、光輝素材的演化與記憶

光輝誕生於太初紀元的創世遺跡，由星辰能量在沉默中凝結而成。

在早期，人們誤將其視為普通的「強化石」。

直到月光女神智姬展開長期觀測與研究，才揭示光輝的真相：

> 它是宇宙語言的碎片，是星辰記憶與意志的容器。

智姬在星辰評議會的《符文語初論》中，首次為這類結晶命名為「符文石」，並留下經典記錄：

> 當星光於沉默中回應思想，語言便在礦脈間覺醒。

### 光輝素材分類

- 光輝碎片：魔獸與遺跡掉落的能量單元，是符文製作的基礎。
- 光輝核心：由碎片提煉的高階結晶，是星紋導通的主要媒介。
- 失去光輝的強化石：早期鍛造殘核，可再經重鑄回復能量。
- 光輝的強化石：中階導通素材，用以重塑符文記憶。
- 光輝的符文石：完成導通後的正式符文，可嵌入裝備，具有星紋意識連結。

所有素材可於狩獵、遺跡與古戰場中獲取，是帕拉維亞世界中最珍稀的能量遺產之一。

## 二、符文進階與曜階鍛造

光輝符文石可安裝於裝備核心槽，其潛能將隨曜階覺醒而展開。

曜階分為五層：

- 聖曜
- 古曜
- 太曜
- 幻曜
- 極曜

### 升階條件

- 兩枚相同曜階符文石。
- 對應曜階升級卷軸。
- 一顆光輝核心。

高階鍛造風險極高，若無保護卷軸輔助，符文可能因能量崩解而失效。

### 曜階效果

- 聖曜：啟動共鳴技能，增幅基礎能力。
- 古曜：強化裝備結構，解鎖部分劇情模組。
- 太曜：突破星紋極限，與遺跡產生交互。
- 幻曜：開啟神性融合模組，對應特定職業劇情。
- 極曜：象徵宇宙原律的臨界，能量導通至最終純度。

## 三、卷軸與鍛造機制

卷軸是導通與鍛造的媒介，分為三類：

- 升級卷軸
- 保護卷軸
- 光輝卷軸

### 光輝的升階卷軸

以顏色區分導通階級：

- 白
- 綠
- 藍
- 紫
- 紅
- 金

導通階級越低，星紋結構越單純，因此越容易成功；導通階級越高，符文承載的星辰記憶越龐大，升階時越容易產生能量排斥。

每階成功率不一，低階最高可達 85%，高階則會明顯下降。

| 導通階級 | 升階成功率 |
| --- | ---: |
| 白 | 85% |
| 綠 | 75% |
| 藍 | 65% |
| 紫 | 55% |
| 紅 | 45% |
| 金 | 35% |

卷軸可於遺跡、古戰場與鍛造坊中取得。

### 光輝的保護卷軸

保護卷軸並非單純提高成功率，而是在升階失敗時嘗試穩定符文核心，防止能量崩解。

依導通階級決定保護成功率與結構保留度。階級越高，符文內部星紋越複雜，保護難度也越高：

| 導通階級 | 保護成功率 |
| --- | ---: |
| 白 | 90% |
| 綠 | 80% |
| 藍 | 70% |
| 紫 | 60% |
| 紅 | 50% |
| 金 | 40% |

使用時需消耗光輝核心，數量為 1 至 5 顆不等。

保護卷軸能在升階失敗時保留符文結構。若保護也失敗，符文不一定完全消滅，但可能降階、裂紋化，或轉化為需要重煉的殘核。

### 光輝卷軸

光輝卷軸由光輝核心 ×10 製作，需擁有高階星紋導通資格並通過星辰評議會的認證。

此卷軸能將符文升階並完整保留詞條，無失敗風險。

使用後，施作者會短暫進入能量休眠，象徵星辰允准與靈魂共鳴的完成。

## 四、光輝神石

當鍛造師收集光輝核心若干與一顆未命名的符文石，可於星紋共鳴台進行「符文鑑定儀式」，生成新的能量結晶：光輝神石。

光輝神石會在智姬的星辰觀測法中被鑑定，展現其真正屬性與潛在詞條。

每一顆神石皆擁有獨一無二的星紋簽印，被稱為「會訴說名字的符文」。

若鍛造失敗，光輝核心會失去導通而化為「無名之核」。這些殘核仍可用於重煉或作為卷軸製作的副素材。

## 五、詞條系統與屬性分類

每枚符文可擁有 1 至 3 個詞條，依顏色稀有度區分：

白（一般）→ 綠（精良）→ 藍（稀有）→ 紫（史詩）→ 紅（神話）→ 金（傳說）。

### 詞條八系

1. 戰鬥強化系：提升攻擊、暴擊、貫穿與破防。
2. 防禦支援系：增幅生命、護盾與抗性。
3. 回復輔助系：強化治癒、能量回流與冷卻縮減。
4. 元素共鳴系：強化火、水、風、土、雷、冰、光、暗屬性。
5. 特效觸發系：觸發燃燒、凍結、麻痺、星痕爆發。
6. 技能強化系：提升職業主動技能與武器特技威力。
7. 增益支配系：延長增益狀態與光環範圍。
8. 劇情世界系：與遺跡、地脈或神祇產生能量共鳴，開啟特殊劇情。

紅階以上的符文可覺醒「星紋意志」，形成微弱靈魂，能在特定條件下自行觸發共鳴反應。

## 六、洗鍊石與重鑄

鍛造師可使用洗鍊石調整或強化符文詞條。

- 基礎洗鍊石：重置單一詞條。
- 高階洗鍊石：可重置兩項並微幅提升稀有度。
- 星紋洗鍊石：由光輝核心提煉，可穩定提升品質。
- 神性洗鍊石：來自女神試煉與古遺跡，可賦予「星靈領域」等特殊效果。
- 共鳴精華：能使多顆符文同步共鳴，短暫提升全裝備性能。

洗鍊象徵鍛造師與符文靈魂的再調和，是對星辰記憶的重編。

## 七、裝備插槽與導通條件

稀有級以上裝備皆設有符文插槽，數量依品質與材質而定。

這些插槽構成裝備能量導通的核心骨架：

- 符文核心槽：安裝光輝符文石，決定主屬性導通。
- 元素輔助槽：可嵌入最多三顆元素石，增強屬性抗性與相性。
- 共鳴擴充槽：安裝劇情模組或高階技能模組，觸發遺跡反應與共鳴事件。

當三種屬性形成相剋組合時，將生成「元素平衡陣」，穩定能量流但削弱爆發力。

### 啟動條件

- 階級需達對應星紋認證。
- 特定劇情儀式（如星辰儀、浮游島祭、智姬觀測）將解鎖更高等導通層級。

## 八、光輝之路的信念

光輝符文，是星辰與靈魂的語言。

每一顆符文的閃耀，皆映照鍛造者的信念與抉擇。

它們記錄的不僅是能量的運行，更是生命在宇宙中留下的迴響。

> 當你洗鍊的不只是屬性，而是信念與星光的軌跡時，
> 那顆符文便會以你的意志閃耀。
