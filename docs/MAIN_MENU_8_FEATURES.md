# 主選單 8 功能規格

重新開發版主選單固定為 8 個主要功能。這 8 個入口會成為後續所有玩法的骨架。

## 1. 新的傳承

用途：開始新遊戲。

流程：

1. 播放父親遺言或序章片段。
2. 選擇五條路線之一。
3. 命名幫派。
4. 建立初始存檔。

## 2. 繼續遊戲

用途：讀取最近一次存檔。

狀態：

- 有存檔：顯示最近 Day。
- 無存檔：按鈕 disabled。

## 3. 父親主線

用途：查看主線章節、父親故事、遺言片段與已解鎖線索。

初期內容：

- 序章：雨夜遺言。
- 第一章：舊帳本。
- 第二章：背叛者的名字。
- 第三章：父親不願碰的生意。

## 4. 幫派基地

用途：進入建設與資源管理。

初期功能：

- 查看金錢、聲望、人力、警戒值。
- 建設收入型、防衛型、情報型設施。
- 升級基地。

## 5. 聯機戰鬥

用途：PVP 與攻擊其他玩家幫派。

初期狀態：

- v0.4 先做入口與規格。
- v0.5 可加入本地模擬玩家。
- 正式聯機需後端儲存與戰鬥驗證。

## 6. 夥伴圖鑑

用途：查看角色、隊伍、信任度、能力與稀有度。

初期功能：

- 顯示已解鎖角色。
- 查看角色技能與背景。
- 設定攻擊隊伍與防守隊伍。

## 7. 榮耀殿堂

用途：成就、戰績、排行榜、重大里程碑。

初期功能：

- 單機成就。
- 父親主線進度。
- PVP 戰績預留。

## 8. 設定

用途：音效、畫面、存檔、帳號、資料管理。

初期功能：

- 音量設定。
- 存檔管理。
- 清除本地資料。
- 未來加入帳號綁定。

## 主選單 UI 建議

- 桌面版：2 欄 x 4 列。
- 手機版：1 欄垂直排列。
- 每個按鈕包含功能名稱與一句短描述。
- 未完成功能可點擊，但顯示開發中提示。
- 主選單底部顯示版本號：`v0.4.0 Redevelopment`。
