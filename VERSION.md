# 黑道建築 Underworld Architect - 版本資訊

## 目前開發目標：v0.4.0 Redevelopment

最後更新：2026-06-16

本版本是重新開發起點，目標是把原本展示型原型整理成可以長期開發的遊戲架構。

## v0.4.0 核心目標

### 1. 主選單重新構建

主選單固定為 8 個功能入口：

1. 新的傳承
2. 繼續遊戲
3. 父親主線
4. 幫派基地
5. 聯機戰鬥
6. 夥伴圖鑑
7. 榮耀殿堂
8. 設定

### 2. 聯機戰鬥規格

v0.4 先建立入口與設計規格，為後續 PVP 做準備。

預定方向：

- 攻擊其他玩家幫派。
- 防守隊伍配置。
- 戰報系統。
- 排行榜預留。
- 先本地模擬，後續再接遠端玩家資料。

### 3. 父親主線大綱

父親故事將作為初始主線核心。

初期章節：

- 序章：雨夜遺言
- 第一章：舊帳本
- 第二章：背叛者的名字
- 第三章：父親不願碰的生意
- 第四章：阿福的沉默
- 第五章：戒指的主人

### 4. GitHub 開發流程

之後開發以 GitHub repository 為主：

- 使用 `git` 管理版本。
- 使用 `gh` 登入、開 PR、查看 repo 狀態。
- 每個功能使用分支開發。
- 大功能完成後再合併到 `main`。

## v0.4.0 驗收標準

- 主選單顯示 8 個功能入口。
- 所有入口都有清楚名稱與狀態。
- 父親主線有初步章節資料與入口。
- 聯機戰鬥有入口與系統規格。
- README、版本文件與設計文件一致。
- 程式碼可以在本機建置。
