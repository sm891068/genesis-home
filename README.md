# 黑道建築 Underworld Architect

`黑道建築 Underworld Architect` 是一款黑道題材的策略經營遊戲。玩家繼承父親留下的勢力，在城市暗流中經營據點、招募角色、處理幫派衝突，並逐步揭開父親當年建立組織的真相。

## v0.1 重建基礎

本專案從 `v0.1 Rebuild Foundation` 重新開始規劃。

這代表過去所有 `0.x` 原型、文件、角色設定與功能想法都會被整合、整理、重新排序，然後從 `v0.1` 作為新的正式開發起點。這不是單一功能更新，而是整個遊戲全面翻新：

- 主選單重建。
- 存檔與讀取重建。
- 遊戲狀態資料重建。
- 角色、女王、秘書、夥伴與後宮/信任系統重建。
- AI 角色思考與玩家對話反應系統納入核心規劃。
- 圖片、素材、設定文件與生成資產都統一放進本保存庫管理。
- 聯機戰鬥與 PVP 先做規格與資料結構，後續逐步實作。

## v0.1 初步主選單

1. 新的傳承：開始新遊戲。
2. 繼續遊戲：讀取最近存檔。
3. 父親主線：查看父親故事章節與主線進度。
4. 幫派基地：進入建設、資源與據點管理。
5. 聯機戰鬥：PVP、攻擊其他玩家幫派、戰報。
6. 夥伴圖鑑：角色、隊伍、信任度與能力。
7. 榮耀殿堂：成就、排名、戰績與里程碑。
8. 設定：音效、畫面、存檔、帳號與資料管理。

## 角色方向

v0.1 會先建立角色分類與資料基礎，後續再擴充大量角色。

預計包含：

- 女王型角色：統御、壓迫感、勢力外交與高階指揮。
- 秘書型角色：管理、財務、情報、任務排程與輔助決策。
- 戰鬥型角色：攻擊、防守、PVP 隊伍。
- 情報型角色：事件預警、弱點分析、談判情報。
- 醫療/支援型角色：恢復、降低損失、提升隊伍續航。

## AI 與互動方向

創世紀之家中的 AI 相關功能會納入新規劃，但會先以規格與資料結構為主。

目標是讓角色不只是固定台詞，而是能根據玩家對話與選擇做出反應：

- 角色記住玩家曾經做過的選擇。
- 角色依照性格、信任度、路線傾向回應玩家。
- 後宮/信任系統不只加數值，也會影響對話態度。
- AI 思考結果會影響建議、事件反應與關係變化。
