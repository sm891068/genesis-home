# Palavia Official Site

《帕拉維亞》官方網站前端原型，包含：

- 官方進入首頁動畫「十三星環啟示」
- 五大界域視覺介紹
- 右上角 4 × 4 漢堡主選單
- 本地備份系統
- 世界觀設定資料

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Local Backup

```bash
npm run backup:init
npm run backup:run
npm run backup:list
npm run backup:verify
npm run backup:prune
```
