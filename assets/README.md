# Assets

本資料夾用來集中管理黑道遊戲的圖片、素材、生成提示詞與素材說明。

## 資料夾

- `characters/`：角色圖片、角色草圖、立繪。
- `backgrounds/`：場景背景，例如雨夜辦公室、街區、基地。
- `ui/`：按鈕、圖示、介面素材。
- `prompts/`：AI 圖片生成提示詞、角色外觀提示詞。
- `generated/`：已生成但尚未正式整理進遊戲的素材。

## 素材規則

每個重要素材都應該保留：

- 用途。
- 對應角色或場景。
- 生成日期。
- 提示詞來源。
- 是否已進入遊戲。

素材先集中管理，再決定是否進入 `public/` 或正式遊戲畫面。
